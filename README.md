# 📚 Amrit Library Website

Amrit Library Website is a modern and responsive library website built using **HTML, CSS, and JavaScript**. The website provides information about the library, its services, book collections, and contact details. It is designed with a clean layout, smooth animations, and a user-friendly interface to make browsing easy for visitors.

This project demonstrates front-end web development skills and can be used as a template for a real library website.

---

## 🌟 Features

- 📖 Modern and attractive homepage
- 📚 Information about library services
- 🏫 About Library section
- 📞 Contact page with library details
- 🎨 Beautiful UI design with CSS
- ⚡ Smooth animations and hover effects
- 📱 Fully responsive design for mobile and desktop
- 🧭 Simple navigation menu

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout design  
- **JavaScript** – Interactive features  
- **Font Awesome** – Icons
